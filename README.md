**Install***

```
https://github.com/Caged/nba-player-tracking.git
npm install
bower install
```

**Sync**

```
./script/syncstats
./script/syncteams
./script/tometricdefs
./script/tocsv > source/players.csv
```

**Start**

```
./script/server
open http://localhost:4567
```
